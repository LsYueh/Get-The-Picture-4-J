# 時間戳記

| COBOL PIC                |   用途   | CLR對應 |  編碼  |  解碼  |
| ------------------------ | ------- | :-------: | :----: | :----: |
| `X(14)` (YYYYMMDDHHmmss) | 時間戳記 | `DateTime` | ✅ | ✅ |

> 其他戳記格式可透過 `X(7)` (yyyMMDD) + `X(6)` (HHmmss) = `X(13)` (yyyMMDDHHmmss) 的方式進行組合
