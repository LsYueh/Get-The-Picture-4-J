# 文字 (Alphabetic/Alphanumeric)
| COBOL PIC   | 說明                   |  CLR對應  |  編碼  | 解碼  | 說明 |
| ----------- | ---------------------- | :---------: | :----: |:----: | :--: |
| `X(n)`      | 任意字元，長度 n            | `string`  | ✅ |✅ | -- |
| `A(n)`      | 只允許字母                 | `string`  | ✅ |✅ | -- |
| `AN(n)`     | 字母 + 數字                | `string`  | ❌ |❌ | 請用 `X(n)` |
| `G(n)`      | 雙位元組字元 (DBCS, EBCDIC) | `string`  | ❌ |❌ | 請用 `X(n)` |
